# just-schedule

| Handler       |
| ------------- |
| function.main |

---

Demonstrates a scheduled function that fetches vehicle count from the Samsara API. Runs at 4 PM on weekdays and 2 PM on weekends. Use this as a starting point for building scheduled reporting or monitoring functions.

### Event Parameters

This function does not require any event parameters.

### Secrets

| Secret Name     | Description                                |
| --------------- | ------------------------------------------ |
| SAMSARA_API_KEY | Samsara API key used to fetch vehicle data |
